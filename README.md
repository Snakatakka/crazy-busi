this is a little bus game i made for HCTG's TransitTime side quest!

# CONTROLS

W & S / Up & Down - Move forwards and backwards. 
-# Note that if you're moving in a certain direction, pressing the opposite direction won't immediately send you in that direction because PHYSICS!
A & D / Left & Right - Turn the bus. 
-# Note that the bus can only be turned if you're actively moving.
